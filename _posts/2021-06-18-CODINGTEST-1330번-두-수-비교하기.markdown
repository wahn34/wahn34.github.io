# 두 수 비교하기

## 문제

두 정수 A와 B가 주어졌을 때, A와 B를 비교하는 프로그램을 작성하시오.



  
### 풀이  
 
```python
def solution():
    a, b = map(int, input().split())
    if a > b: print('>##')
    elif (a < b): print('<')
    elif (a==b): print('==')
    else: print('error')
solution()
 ```
